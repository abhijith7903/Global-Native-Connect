# Global Native Connect

## Description
Global Native Connect is a web-based platform designed to enable users from different regions to interact and communicate with native language support. The system focuses on improving cross-cultural communication through a responsive and scalable web interface.

This project was developed as a team-based academic project and later published in an international journal.

## Key Features
- Native language-based user interaction
- Responsive and user-friendly interface
- Backend database integration for data handling
- Optimized backend interactions for improved usability
- Scalable architecture for future enhancements

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Project Details
- Team Size: 4
- Duration: 6 Months

## How to Run the Project
1. Clone the repository: git clone https://github.com/abhijith7903/Global-Native-Connect.git
2. Install XAMPP and start Apache & MySQL
3. Import the database into phpMyAdmin
4. Place the project folder inside the `htdocs` directory
5. Open the project in a browser using `localhost`
